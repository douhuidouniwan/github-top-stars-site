知识型仓库占据主导

前十名里有大量项目不是传统软件，而是课程、书单、路线图和精选列表。这说明 GitHub 对很多开发者来说不仅是代码托管平台，也是学习入口。

可执行的学习路径最有传播力

developer-roadmap、coding-interview-university、system-design-primer 等项目的共同点，是把复杂能力拆成清晰路径，降低了自学和求职准备的门槛。

长期维护比短期热点更重要

这些项目多数经历多年迭代，靠持续更新、社区贡献和稳定价值累积 star。它们的成功更像基础设施，而不是一次性爆款。
